# DermaSense AI

AI-powered multimodal dermatology platform for precision skin analysis.

## Overview

DermaSense AI is a research and development platform that integrates:

- Computer Vision
- EMG
- Electrical Impedance Spectroscopy (EIS)
- Machine Learning
- Clinical Decision Support

to assist dermatological assessment and personalized treatment planning.

## Technologies

- Python
- PyTorch
- OpenCV
- NumPy
- Pandas
- Streamlit / Flask (if applicable)

## Current Status

This repository contains the research prototype currently under active development.

## Disclaimer

This software is intended for research purposes only and is not approved for clinical use.
